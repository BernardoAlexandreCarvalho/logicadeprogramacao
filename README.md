# 🧠 Lógica de Programação (Portugol)

Repositório dedicado ao meu desenvolvimento de raciocínio lógico utilizando a pseudolinguagem Portugol.

## 🚀 Sobre o projeto

Este repositório documenta a minha jornada nos fundamentos da lógica de programação, utilizando Portugol para compreender os conceitos essenciais antes da transição para linguagens de programação.

O foco está na construção de uma base sólida em algoritmos e resolução de problemas.
