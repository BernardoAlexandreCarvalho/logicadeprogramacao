# Salário Funcionário

Programa que calcula o salário de um funcionário com base na quantidade de dias trabalhados no mês

## Objetivo

Praticar cálculos com base em tempo de trabalho e valor por hora.

## Regras utilizadas

8 horas de trabalho por dia
55,00 por hora trabalhada

## Exemplo de entrada

dias trabalhados: 20

## Exemplo de saída

Salário: R$8800
