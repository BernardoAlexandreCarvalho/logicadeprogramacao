# Conversão de Medidas

Programa que lê uma distância em metros e converte para outras unidades de medida.

## Objetivo

Trabalhar com conversões de unidades e múltiplas operações matemáticas.

## Conversões realizadas

Quilômetros (km)
Hectômetros (hm)
Decâmetros (dam)
Decímetros (dm)
Centímetros (cm)
Milímetros (mm)

## Exemplo de entrada

Digite uma distância em metros: 185.72

## Exemplo de saída

0.18572KM 
1.8572HM 
18.572DAM  
1857.2DM 
18572.0CM  
185720.0MM
