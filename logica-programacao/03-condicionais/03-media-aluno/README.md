# Média Aluno

Programa que lê o nome de um aluno e suas duas notas, calcula a média e informa se ele teve um bom aproveitamento.

## Objetivo

Praticar cálculos com números reais e uso de estruturas condicionais (se/senão).

## Regra utilizada

Média mínima para bom aproveitamento: 7.0

## Fórmula

Média = (nota1 + nota2) / 2  

## Lógica

Se média >= 7.0 → bom aproveitamento  
Caso contrário → desempenho insuficiente  

## Exemplo de entrada

Nome: João  
Nota 1: 8.0  
Nota 2: 7.5  

## Exemplo de saída

Média: 7.75  

## Exemplo de entrada

Nome: Maria  
Nota 1: 5.0  
Nota 2: 6.0  

## Exemplo de saída

Média: 5.5
