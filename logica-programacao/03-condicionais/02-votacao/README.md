# Votação

Programa que lê o ano de nascimento de uma pessoa, calcula sua idade e informa se ela pode votar.

## Objetivo

Praticar estruturas condicionais (se/senão) e cálculos com datas.

## Regra utilizada

Considera o ano atual para calcular a idade  
No Brasil, o voto é permitido a partir dos 16 anos  

## Fórmula

Idade = ano atual - ano de nascimento  

## Lógica

Se idade >= 16 → pode votar  
Caso contrário → não pode votar  

## Exemplo de entrada

Ano de nascimento: 2000  

## Exemplo de saída

Pode votar  


## Exemplo de entrada

Ano de nascimento: 2012  

## Exemplo de saída

  
Não pode votar.
